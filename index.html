Google v. Oracle and API analogies 2021-04-16

Last weekend, to kill time, I decided to read the judgement in the recently concluded Oracle vs Google lawsuit. If you, as a programmer, do not know what this is about, here is a gist: Google re-implemented Oracle’s Java SE/ME APIs to create a runtime for Android (called Dalvik and a later iteration called ART). Oracle promptly filed a lawsuit claiming damages. The latest judgement from the U.S. Supreme court, which was delivered on April 5th, reversed an earlier lower court’s opinion and decided in favour of Google.
